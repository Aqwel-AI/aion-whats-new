# aion-whats-new

![Aqwel-Aion v0.1.8 is out – update now!](Aqwel-Aion%20v0.1.8%20is%20out%20%E2%80%93%20update%20now!.png)

**What's New in Aqwel-Aion v0.1.8** — A Jupyter notebook with new features, algorithms, visualization, examples, and migration guide.

---

## What's in this repo

- **`Whats_New_in_Aion_0.1.8.ipynb`** — Full guide to v0.1.8: new packages (`aion.algorithms`, `aion.visualization`), function reference, runnable examples, troubleshooting, and Aqwel AI info.

---

## Contents of the notebook

- **Version & changelog** — v0.1.8 metadata and dependency notes  
- **Algorithms** — Search (binary, lower/upper bound, jump, exponential, ternary, interpolation, rotated, peaks), array utilities (flatten, chunk, sliding window, rolling sum, pairwise, etc.), graphs (BFS, DFS, toposort)  
- **Visualization** — 1D plots (array, histogram, scatter, running mean, CDF, etc.), 2D/matrix (heatmap, confusion matrix, correlation, attention map), training (history, train vs val, early stopping)  
- **Examples** — Count with lower/upper bound, toposort cycle detection, end-to-end workflow, save plot to file  
- **Extra** — Troubleshooting & FAQ, migration from v0.1.7, Aion vs standard library, links & resources, optional install extras, cheat sheet  
- **Aqwel AI** — License, support, and contact

---

## Requirements

- **Python** 3.10+  
- **aion v0.1.8** — Install with `pip install -e "aion(v0.1.8)/"` or add the path in the notebook  
- **numpy**, **matplotlib** — For visualization (or install `aqwel-aion[full]`)

---

## How to run

1. Clone the repo and open the notebook in Jupyter or VS Code.  
2. Run the setup cell so `aion` is on your path (or install the package).  
3. Run cells in order.
